Padding and Units

Padding is the space between the border and the content area.

Margin is the space outside of the border.

Display controls how an entity is viewed by the browser.

The multiple types of entities are :

  1. Block

    Width, Height, Padding and Margin are respected

    This is a link This is another link

  2. Inline

    Width, Height are Ignored

    This is an H4

    This is another H4

  3. Inline-Block

    Behaves like an inline level entity, but block-level properties are respected

    This is span This is another span

The 2 most common types of relative units are :

  • Em

    Is a relative unit that relates to some other value, usually of the parent value, sometimes of the element itself

  • Rem

    Is a relative unit that also relates to some other value, but only the root elements value.
